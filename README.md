# JAN-KEN-PON - Rock Paper Scissors Game 🪨📄✂️

- JAN-KEN-PON is a modern and interactive version of the classic **Rock-Paper-Scissors** game, inspired by **JanKenUP!** developed by independent game developer **Humita Games**.

This project adds a fun arcade-style experience with AI opponents, animated gameplay, sound effects, customizable settings, and a responsive interface for desktop and mobile devices.

## 🎮 How to Play

1. Open the game in your browser.
2. Click **PLAY** on the menu screen.
3. Select your preferred AI difficulty:
   - 🟢 Easy
   - 🟡 Medium
   - 🔴 Hard
4. Choose your move:
   - 🪨 Rock
   - 📄 Paper
   - ✂️ Scissors
5. Watch the countdown, reveal the computer's choice, and see the result.
6. The scoreboard updates automatically.
7. Customize your experience using:
   - 🎵 Music toggle
   - 🔊 Volume slider
8. First player to reach **10 points** wins the match.

# ✨ Features

### 🕹 Game Features
- Classic Rock-Paper-Scissors gameplay.
- AI opponent with multiple difficulty levels.
- First-to-10 scoring system.
- Automatic score tracking.
- Persistent game settings using `localStorage`.

### 🤖 AI Difficulty Levels
- **Easy** - Random computer choices.
- **Medium** - Balanced AI behavior with basic prediction.
- **Hard** - More challenging AI that adapts to player patterns.

### 🎨 UI & Animations
- Interactive menu screen.
- Animated hand movements.
- Countdown sequence before each round.
- Smooth result transitions.
- Winner/loser popup messages.
- Animated music equalizer.

### 🎵 Audio Features
- Background music support.
- Music enable/disable toggle.
- Real-time volume adjustment.
- Persistent audio preferences.

### 📱 Responsive Design
- Optimized for:
  - Desktop
  - Tablet
  - Mobile devices
